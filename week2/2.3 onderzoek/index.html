Opdracht 2.3: Onderzoeken

Waarom noem je de eerste HTML-pagina: index.html?

Elke webserver is standaard zo ingesteld dat wanneer de domeinnaam van een website in de browser word ingetypt, dat dan de webpagina met de naam index.html getoond wordt.

Met welke HTML versie werken we nu?

HTML5

Met welke CSS versie werken we nu?

CSS 3

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera.)

Verschillende browsers interpreteert source code zoals HTML of CSS op een andere manier waardoor er klein verschil kan ontstaan om hoe het eruit komt te zien, maar voor de functionaliteit van de website zou dit niet moeten uitmaken.
